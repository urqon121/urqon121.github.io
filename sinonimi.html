Razlučivanje sinonima

Za ovo poglavlje krivi su većinom mediji koji su koristili sva tri pojma kao da su istoznačni.

  • Darknet

    • Nije skupina web stranica, već vrsta mrežne tehnologije. Na darknet-ove se spajamo koristeći nestandardne protokole. Obično se koristi istorazinski (engl. peer-to-peer) tip konekcije.

  • Dark web

    • Dark web je dio World Wide Weba koji koristi darknet tehnologiju. Dark web je maleni dio deep weba.

  • Deep web

    • Dio World Wide Weba koji nije indeksiran od strane standardnih web tražilica. Primjer nečega što jest deep web, ali ne i dark web jest bilo što što stoji iza nekakve lozinke (npr. detalji o bankovnom računu, Facebook); većina korisnika svakodnevno koristi deep web.
